Large Deviations (1)

  • Large Deviations
Mathematic
Contents
  1. Cramér定理

Large Deviation 是一个研究罕见事件事发概率为指数型的估计理论，现已成为为概率论的主流分支之一。

Cramér定理

已知$X,X_1,X_2, \ldots ,X_N$是独立同分布的随机变量，那么他们的平均值${M_N} = \frac{1}{N}\sum\limits_{i = 1}^N X_i$满足
$$\lim \limits_{N \to \infty } \frac{1}{N} \ln P( M_N > x ) = - I( x )$$
其中，$I( x )$称为速率函数、Cramér函数或者熵函数。
对于很大的$N$ ，有
$$\ln P( M_N > x ) \approx \exp [- N I( x )]$$
如果我们已知 的概率分布，我们能够通过共轭函数来计算
$$ I(x)=\sup \limits_{\theta >0} {\{ \theta x -\lambda ( \theta ) \} }$$
其中，$\lambda ( \theta )=\ln E[exp ( \theta X)]$ ，即X的累积分布函数。

举例

以抛硬币实验为例，掷正面和反面的概率均为1/2，分别记为$X=0,1$。那么
$$\lambda ( \theta )=\ln E[exp ( \theta X)] = \ln ( 1+ e^\theta) -ln2 $$
通过计算，可得
$$I(x)= x \ln x + (1-x) \ln(1-x) +ln2 $$
进而求得结果。

应用

最早的大偏差理论是由Cramér在保险业务中应用的。因为保险理赔是一个随机事件，为了保证保险公司每月盈利，保险金额的制定成为了一个难题。主要问题是使得每月收到的保险金总额大于理赔总额，换句话说可以让保险公司不盈利的概率随着用户的增加指数级下降。抽象成数学问题，设每月每人保险金额为$q$，缴纳人数为$N$，每个人理赔的金额$X_i$服从独立同分布。通过研究$X$的概率分布，设定不盈利的概率$\eta $，通过求解
$$\ln P( M_N > x ) \approx \exp [- N I( x )] = \eta $$
就能够完美解决这一问题。
