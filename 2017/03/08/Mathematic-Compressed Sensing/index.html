Compressed Sensing and Gelfand Widths

  • Compressed Sensing
Mathematic

The author tries his best to present the basic relationship among Exact Reconstruction, Null Space Property(NSP), Restricted Isometry Property(RIP) and Gelfand Widths.

Introduction and Definitions

Compressed Sensing deals with the problem of recovering a $k$-sparse signal $x \in R^n$ from $m$ linear observations $y \in R^m$, that is
$$ y=Ax,$$ where $A \in R^{m \times n}$ is the sensing matrix.

Usually, Reasearchers use Basic Pursuit(BP) to solve this problem, that is
$$ \min_{t \in R^n} ||t||_1 ~~~s.t. ~~At=y
$$

Exact Reconstruction For any $k$-sparse signal $x$, if there is a unique solution $x$ satisfies
$$ \min_{t \in R^n} ||t||_1 ~~~s.t. ~~At=Ax,$$then we can say $A$ satisfies Exact Reconstruction by BP.

Null Space Property(NSP) $\forall h \in \ker A, h \neq 0, \forall I \subset \{1,2,\ldots,n\}, |I| \le k $, one has $||h_{I}||_1 < ||h_{I^c}||_1$.

Restricted Isometry Property(RIP) For all $k$-sparse signal $x$, there exists a constant $\delta_k \in (0,1)$ such that
$$(1-\delta_k)||x||_2^2 \le ||Ax||_2^2 \le (1-\delta_k)||x||_2^2.$$

$m$-th Gelfand Width of $T$ Let $T$ be a bounded subset of a normed space $E$. Let $m \ge 0$ be an integer. The $m$-th Gelfand Width of $T$ is defined as
$$d^m(T,E):=\inf _S \sup _{x \in S \cap T} ||x||_E,$$ where $||\cdot||_E$ denotes the norm of $E$ and the infimum is taken over all linear subspaces $S$ of $E$ of codimension less than or equal to $k$.

Properties

  • If $\delta_{2k}(A) < \sqrt{2}-1$, then $A$ satisfies Exact Reconstruction.
  • If $rad (\ker A \cap B_1^n) < \frac{1}{2\sqrt{k}}$, then $A$ satisfies Exact Reconstruction.
    We know $\ker A >= n-m$. With the increase of sample size $m$, $rad (\ker A \cap B_1^n)$. Therefore, for a fixed sparsity $k$, the more sample size, the better the performance. At the same time, for a fixed sample size, the sparser the signal, the better the performance.
  • If $d^m(B_1^n,l_2^n) < \frac{1}{2 \sqrt{k}}$, then there exists $A \in R^{m \times n}$ such that
    $$rad (\ker A \cap B_1^n) < \frac{1}{2\sqrt{k}},$$that is, $A$ satisfies Exact Reconstruction.

Relationship

NSP is equivalence to Exact Reconstruction, and hence if we can prove that a sufficient condition can derive NSP, we can conclude that the condition satisfies Exact Reconstruction. RIP and Gelfand Width are the sufficient condition for NSP.
RIP is very easy to understand and therefore is applied broadly to the proof of compressed sensing. Gelfand Width can be used in not only Exact Reconstruction but also Approximate Reconstruction, which means $y=Ax+w$, where $w$ is a random noise.

Tools

Researchers develop different tools for different sensing matrix $A$. Sensing matrix includes unstructured matrix such as gaussian matrix and bernoulli matrix, and structured matrix such as partial Fourier matrix and Walsh matrix. $\epsilon$-net argument and chaining techniques are proposed to deal with unstructured matrix for compressed sensing. Empirical processes are used to deal with structured matrix.

Recommended Books

  1. D. Chafa\̈i, O. Guédon, G. Lecué, and A. Pajor, Interactions between compressed sensing random matrices and high dimensional geometry. Société Mathématique de France, 2012.
  2. R. Vershynin, High Dimensional Probability An Introduction with Applications in Data Science, Draft, 2017.
